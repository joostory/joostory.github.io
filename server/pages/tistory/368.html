Ubuntu 12.04 countdown banner

2012. 4. 3. 22:37

Ubuntu는 새버전이 출시될 때마다 countdown banner를 만들어왔다.
계속하면 문화가 된다고 했던가? 이미 Ubuntu 에서는 countdown banner로 디자인 공모 비스무레한 것을 하고 있었다.

이번 12.04에는 무려 13개나 되는 작품이 나왔다.
대표로 3개가 선발되기는 했지만 그 중 하나만 쓰기엔 나머지가 너무 아까워서 랜덤으로 나오도록 만들어봤다.

<a href="http://www.ubuntu.com" id="ubuntu_link"></a>
<script>
(function(count) {
  var random = parseInt(Math.random() * count) + 1;
  var url = "http://www.ubuntu.com/countdown/banner" + random + ".png";
  var countdown = new Image();
  countdown.onload = function() {
    this.style.display = "";
  };
  countdown.alt = "우분투 다음 판이 곧 나옵니다.";
  countdown.src = url;
  countdown.style.display = "none";
  var ubuntuLink = document.getElementById("ubuntu_link");
  ubuntuLink.appendChild(countdown);
})(13);
</script>

복사해서 붙여넣기하면 바로 사용가능 :)
