tomcat 실행 시 java option 설정

2009. 2. 26. 18:47
지금까지 tomcat을 실행할 때 eclipse가 아닌 상태에서 실행한 적이 거의 없는 것 같습니다.
콘솔에서 tomcat을 실행하려니 정말 짜증나났(이제 안나니까 ^^)네요. :)

eclipse에 tomcat plugin을 설치하고, 실행하면 거의 뭔가를 만질 일이 없었는데...
직접 tomcat을 실행하려니 server.xml 에 Context 등록해야지~ java 옵션 설정해야지~ tomcat의 까만창 떠있어야지~ -_-;;
어쨌거나 다른건 다 괜찮은데... java 옵션은 어디서 설정하는 것인지 전혀 몰랐기에...
거의 하루를 보냈네요. 이거 하느라.. ㅠㅠ

tomcat 의 java 옵션을 설정하는 방법은 다음과 같습니다.

1. eclipse (Windows, Unix 모두 가능)
eclipse 에서는 뭐... 그림만 봐도 금방.. :)


2. JAVA_OPTS 설정
JAVA_OPTS 라는 환경설정 변수에 원하는 값을 넣어주면 됩니다.

2-1. Windows
tomcat을 실행하는 파일을 따로 만들었는데 그림과 같은 형식으로 만들었습니다.
2-2. Unix
unix 에서도 Windows랑 비슷한데 shell이 다르니.. :)
JAVA_OPTS=-Dtest -Dfile.encoding=euc-kr
export JAVA_OPTS

3. Tomcat 실행 프로그램을 사용 (Windows 만 가능)
이게 가장 좋은 방법인 것 같습니다. Windows 에서는..
저렇게 쉽게 java option을 설정하고, 실행도 깔끔하게 할 수 있는데... 콘솔에서 한다고 고생을 왜 했는지.. ㅠㅠ
(아! 사실 이 방법을 찾아본 이유가.. 메신저용으로 사용 중인 컴퓨터에 서버를 띄어놓으면 좋겠다고 생각해서
tomcat만 딱 windows에 띄어놓고, 작업한 것을 여기다가 옮겨놓으면 작업컴퓨터는 빠르게 작업할 수 있어서 좋고,
테스트는 작업에 상관없이 할 수 있고, 어차피 놀고 있던 컴퓨터 잘 활용해서.. 여러가지로 좋겠다는 생각에 시작한 것이라 이 방법이 가장 좋다고 한 것입니다. tomcat을 실행하는 컴퓨터가 unix계열이라면 2-2 가 최고겠죠.)
