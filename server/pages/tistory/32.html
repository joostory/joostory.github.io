구글 검색 역시 좋다.

2008. 2. 25. 22:29
웹페이지를 만들다가 select 를 multiple로 만들고 전부 선택해서 그걸 넘겨받고 싶었다.
어떻게 검색을 해야할지 몰라서 select multiple 이라고 검색을 했다.

대부분이 select 에 대한 설명이고 php 로 처리하는 방법도 많이 나왔다.
그러나 내가 원하는 것은 java를 통해서 이 것을 처리하는 방법을 원했던 것이다.
php에서 select의 이름을 배열형태로 넘겨주는 것이 java에서도 통하리라고는 생각하지 않았다.


그래서 select multiple java 라고 검색을 했다.

결과가 또 별로 좋지 않았다. 이번엔 java를 활용하기는 하지만 대부분 DB에서 select 를 할 때 multiple 한 값을 처리하는 방법들이 대다수를 이루었다.
게시판 검색결과는 좀 근접했지만 방법이 아닌 질문이다. _-;

그런데!!! 마지막에 있던 Google 검색결과의 첫번째 결과는 아주아주아주 정확하게 내가 원하는 값을 가져다 주었다.
오호~ 놀라워라~ 마치 이것을 원하고 있었다는 것을 안다는 듯이..
구글 검색은 역시 좋다...
