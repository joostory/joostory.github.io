Ubuntu 10.10 설치해봤습니다.

2010. 9. 3. 23:31
Ubuntu 10.10 의 개발버전 배포가 시작되었습니다.
열혈 Ubuntu Lover가 여기에 빠질 수 없어 금요일 밤에 upgrade를 시작했습니다.


두근거리는 마음을 진정시키며 업그레이드를 했고, 시간은 1시간 정도 소요되었습니다.
그리고 역시나 몇가지 문제가 발생했습니다.


1. "modprobe vboxnetflt failed" 발생

X가 뜨기도 전에 이런 메시지가 나오면서 멈췄습니다.
전에도 vbox 관련된 문제를 겪지 않았더라면 오늘 안에 이 문제를 해결할 수 없었을겁니다.
방법은 간단합니다.

$ sudo /etc/init.d/vboxdrv setup



2. xorg.conf 문제

이 문제는 생각보다 간단합니다.
자세한 이유는 모르지만 Ubuntu는 더이상 X.org 설정파일을 사용하지 않습니다.
제가 전에 실수로 만들어놔서 생긴문제였습니다. 삭제하면 해결!

$ sudo rm /etc/X11/xorg.conf



3. nvidia driver 문제

1, 2 번 문제를 해결했는데 이젠 해상도가 이상하네요.
드라이버가 뭔가 안 맞는 것 같았습니다. 녹색불이 들어와 있었지만 그래도 지우고 다시 설치하니 잘 됩니다.



4. Eclipse의 subversive 동작하지 않음

Ubuntu에서 소프트웨어 설치를 통해서 eclipse를 설치했었는데... 이상하게 Subversive가 안되네요.
이제 Helios 다운로드 받아서 쓸때가 되었다고 생각했습니다. 이건 뭐.. 저에게만 일어나는 문제이니.. ^^



* 좋아진 점 몇가지

위에 나열한 4가지의 문제를 제외하고서는 특별한 문제는 겪지 못했습니다. 아직까진.. ^^
좋아진 점도 몇가지 있는데요. 대표적인 것이 테마입니다.
다른 테마로 바꿀 생각조차 들지 않는 좋은 테마입니다.
