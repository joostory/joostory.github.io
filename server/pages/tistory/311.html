왜 사무실에서는 창의적인 일을 하지 않을까?

2011. 4. 28. 21:57

대부분이 공감할 것 같은 이야기.
사무실에서는 정말 창의적인 일들이 잘 일어나지 않는다.
주로 집에 가는 길 차안에서.. 퇴근 후에 소파에 늘어져 있다가.. 화장실에 앉아있다가..
이럴때 재미난 생각들이 많이 나는 것 같다.

그래.. 책상에 앉아서인 것 같다.
사무실을 마음대로 꾸밀 수 있도록 해야한다. 그래야 창의적으로 일할 수 있을 것 같다.
