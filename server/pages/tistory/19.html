2008. 1. 25. 17:04

Eclipse의 Code Assist가 동작하지 않았다.
심지어 Window > Preferences > General > Keys 에 Code Assist 조차 없었다. -_-;;
단지 있는 것은 Context Assist 뿐이었다.
