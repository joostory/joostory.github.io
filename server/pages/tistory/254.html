Google reader에 Sent to Yozm 설정하기

2010. 5. 21. 22:50
이런 저런 이야기를 하기에 앞서서 "요즘" 을 아시나요?
모르신다면 http://yozm.daum.net 을 한번 가보신 후에 아래에 설명할 방법을 보시는 것이 좋을 것 같습니다. 간략히 설명드리면 yozm은 Daum에서 10대를 타겟으로 시작한 SNS 입니다. 아기자기한 프로필과 디자인, Facebook이 짭잘한 수입을 올리고 있는 SNG를 일부 도입하고 있습니다.

오늘 Google reader를 이것저것 만져보다가 Send to 를 발견했습니다. (이제야.. ^^;;)
Twitter, Facebook 정말 많은 서비스들에 보내기를 할 수 있더군요.
그러다 문득 얼마전에 발견한 yozm api를 적용해보기로 했습니다.
(Daum 여기저기에 붙어있는 "yozm에 보내기" 버튼 링크를 유심히 보다가 알게된 거죠.)

API는 다음과 같은 형태입니다. 잘 모르는 것에는 설명을 붙이지 않았습니다.

API URL : http://yozm.daum.net/api/popup/post?prefix=[제목]&link=[URL]&sourceid=[SOURCE_ID]&meta=&key=&imgurl=[IMG_URL]&crossdomain=1&callback=

Send to의 custom link에는 다음과 같이 써주면 됩니다.
sourceid는 출처를 말하는데 code화되어 있습니다. 안쓰면 웹에서 쓴걸로 표시합니다. (알려주셔서 감사합니다. ^^)

Name : Yozm
URL : http://yozm.daum.net/api/popup/post?prefix=${title}&link=${url}&sourceid=&meta=&key=&imgurl=&crossdomain=1&callback=
Icon URL : https://t1.daumcdn.net/cfile/115D0A164B0B73BC33


설정이 완료되면 Send to에 Yozm이 추가됩니다.


보내기를 클릭하면 새창으로 이런 페이지가 나오고 확인을 클릭하면 Yozm에 등록됩니다.
(popup 용으로 만들어진 페이지라서 좀... ^^;; 이상하게 보이죠? 그래도 잘 동작하니까..)


Yozm에는 api가 있는 것 같기는 한데... 아직 공개가 안되어 있는 것 같습니다.
어서 공개해주었으면 좋겠네요. :)
