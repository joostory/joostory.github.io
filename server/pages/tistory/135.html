Ubuntu 삽질기

2008. 11. 27. 01:44
Ubuntu 8.10에서는 아직도 OpenOffice.org 2.4 버전을 쓰고 있습니다.
3.0이 나온지가 언젠데... 그래서 얼마전부터 3.0을 launchpad 에서 받아서 쓰고 있었습니다.
오늘.. 아니.. 어제 아침 OpenOffice 관련 update 가 있었는데... 그 뒤로 갑자기 실행이 되지 않았습니다. -_-;;
그래서 2.4 로 다운그레이드 하려니 language-support-ko 이런 것 까지 지우려드는 겁니다.
아... 정말...

결국 야근을 결심하고.. 오후 7시가 넘어서부터 Ubuntu 8.10 을 다시 설치하기 시작했습니다.
제가 Ubuntu 만 1년동안 몇번을 설치했는지 모릅니다. 당근 후딱후딱 설치했죠.
파티션은 간단히 /, /home, swap, fat32(/media/sda4) 의 4개로 나누어 쓰고 있는데
재설치시에는 / 만 재설치하면 설정도 그대로 남아 있어 아주 편리합니다.

그런데....
설치가 완료되고 나서 부팅이 안되는 겁니다.
grub 콘솔이 뭘 원하는지 껌뻑거리고 있었습니다. -_-;;

그래서 / 를 완전히 포맷하고 다시 설치.. 이제는 부팅이 잘 되더군요.

그런데 또!!! driver 설치를 하는데 갑자기 모든 시스템이 멈추는겁니다.
키가 아무것도 먹히지 않더군요. 그래서 좀 찜찜하지만 전원을 껐다가 켰습니다.
그랬더니만 fsck 를 직접 쳐서 고쳐달라고 root 콘솔이 깜빡입니다.
몇번을 그랬는지 모릅니다. (전원을 그냥 꺼서 그런 듯...)
설치를 몇번이나 해봤지만 이런 상황이 되니 데이터가 날아가는건 아닌지 걱정이 되기 시작했습니다. ㅠㅠ

다행히 다시 부팅되었을때는 정상적으로 동작하더군요. 데이터도 말짱하고.. 휴...

이렇게 설치가 끝나는구나 했지만 문제는 여기서 그친게 아니었습니다.
폰트가 갑자기 볼드체만 인식하고.. flash player 버전이 낮아서 그런지 flash가 잘 안보이고..
AWN 이 맛이 갔는지 멀쩡히 등록된 OpenOffice 가 보이질 않고...
PDT 는 왜 그러는지 다운로드 속도가 20kb/s 이고..
emacs 설정을 왜 /etc/emacs 에 했었는지.. ㅠㅠ 다 날아갔고...
eclipse 아이콘 파일은 리눅스용으로 제공안하는지 아이콘이 엉망이고...
